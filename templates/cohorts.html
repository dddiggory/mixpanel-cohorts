Mixpanel Cohorts

Diggory Rycroft, June 2016


This is a free, unofficial service for easily converting a filtered set of Mixpanel user profiles, like this:

... into a saved ad hoc cohort, accessible in both your Engagement reports (e.g., Segmentation, Funnels, Retention) and for easy Notification targeting (i.e., saving a complex set of filters and using them for many different Notifications).

To use it, we'll simply set up a Webhook notification in Mixpanel.
This can be either a one-time grouping from the Explore tab, or a recurring check via the Notifications tab.

Direct your webhook to the following URL, replacing the bold portions with your preferred values:

https://mixpanel-cohorts.herokuapp.com/cohorts?cohort=Cohort+Name&token=Your+Project+Token

Be sure to use "+" instead of spaces; the service will convert this back to spaces before sending the cohort to Mixpanel. Within seconds, your targeted group will have a new list-type property on their user profiles, called "Cohorts". You can use this to run ad hoc analyses on groups you are interested in. Enjoy!
